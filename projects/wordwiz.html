Wordle Wizard

Helps you find the perfect wordle guesses

Project Description

This useful app uses your Wordle guesses to generate the best possible guesses for the next round. All the user has to do is input what they guessed and what the results were, and the app will show suggested guesses along with all of the possible answers. The app is designed to mimic the actual Wordle game's style. Built entirely in Google's Angular framework, it is a simple, easy-to-use, and responsive website. In the near future, I plan to add a login feature to save user's guess history and track how much information is gained from each guess, allowing users to see their most frequent guesses and how helpful they are.
